### ✨ Δυνατότητες

- 🏎️ **Ελαφρύ περιβάλλον εκτέλεσης:** Οι κοινές ροές εργασίας CLI και κατάστασης εκτελούνται με ελάχιστη μνήμη σε εκδόσεις release.
- 💰 **Αποδοτική ανάπτυξη:** Σχεδιασμένο για πλακέτες χαμηλού κόστους και μικρά cloud instances χωρίς βαριές εξαρτήσεις.
- ⚡ **Γρήγορη εκκίνηση:** Το περιβάλλον εκτέλεσης Rust σε ένα ενιαίο δυαδικό αρχείο διατηρεί την εκκίνηση εντολών και δαιμόνων σχεδόν ακαριαία.
- 🌍 **Φορητή αρχιτεκτονική:** Μια ενιαία ροή εργασίας σε ARM, x86 και RISC-V με εναλλάξιμους παρόχους/κανάλια/εργαλεία.

### Γιατί οι ομάδες επιλέγουν το ZeroClaw

- **Ελαφρύ από προεπιλογή:** μικρό δυαδικό αρχείο Rust, γρήγορη εκκίνηση, χαμηλό αποτύπωμα μνήμης.
- **Ασφαλές εκ σχεδιασμού:** σύζευξη, αυστηρό sandboxing, ρητές λίστες επιτρεπόμενων, περιορισμός στον χώρο εργασίας.
- **Πλήρως εναλλάξιμο:** τα βασικά συστήματα είναι traits (πάροχοι, κανάλια, εργαλεία, μνήμη, σήραγγες).
- **Χωρίς εγκλωβισμό (No lock-in):** υποστήριξη παρόχων συμβατών με OpenAI + δυνατότητα προσθήκης προσαρμοσμένων σημείων τέλους (endpoints).

## Στιγμιότυπο δοκιμής επιδόσεων (ZeroClaw vs OpenClaw)

Γρήγορη δοκιμή σε τοπικό μηχάνημα (macOS arm64, Φεβρουάριος 2026) με κανονικοποίηση για υλικό edge 0.8GHz.

| | OpenClaw | NanoBot | PicoClaw | ZeroClaw 🦀 |
|---|---|---|---|---|
| **Γλώσσα** | TypeScript | Python | Go | **Rust** |
| **RAM** | > 1GB | > 100MB | < 10MB | **< 5MB** |
| **Εκκίνηση (0.8GHz core)** | > 500s | > 30s | < 1s | **< 10ms** |
| **Μέγεθος δυαδικού** | ~28MB (dist) | N/A (Scripts) | ~8MB | **~8.8 MB** |
| **Κόστος** | Mac Mini $599 | Linux SBC ~$50 | Linux Board $10 | **Οποιοδήποτε υλικό $10** |

> Σημειώσεις: Τα αποτελέσματα του ZeroClaw μετρήθηκαν σε εκδόσεις release χρησιμοποιώντας `/usr/bin/time -l`. Το OpenClaw απαιτεί το περιβάλλον εκτέλεσης Node.js (συνήθως ~390MB πρόσθετη επιβάρυνση μνήμης), ενώ το NanoBot απαιτεί Python. Τα PicoClaw και ZeroClaw είναι στατικά δυαδικά αρχεία. Τα παραπάνω σχήματα RAM αφορούν τη μνήμη κατά την εκτέλεση. Οι απαιτήσεις μεταγλώττισης είναι υψηλότερες.

### Αναπαραγώγιμη τοπική μέτρηση

Οι ισχυρισμοί των δοκιμών ενδέχεται να αλλάξουν καθώς ο κώδικας εξελίσσεται, επομένως μετράτε πάντα την τρέχουσα έκδοση τοπικά:

```bash
cargo build --release
ls -lh target/release/zeroclaw

/usr/bin/time -l target/release/zeroclaw --help
/usr/bin/time -l target/release/zeroclaw status
```

Παράδειγμα δείγματος (macOS arm64, μέτρηση στις 18 Φεβρουαρίου 2026):

- Μέγεθος δυαδικού release: `8.8M`
- `zeroclaw --help`: περίπου `0.02s` πραγματικός χρόνος, ~`3.9MB` μέγιστο αποτύπωμα μνήμης
- `zeroclaw status`: περίπου `0.01s` πραγματικός χρόνος, ~`4.1MB` μέγιστο αποτύπωμα μνήμης

## Προϋποθέσεις

<details>
<summary><strong>Windows</strong></summary>

#### Απαιτείται

1. **Visual Studio Build Tools** (παρέχει τον MSVC linker και το Windows SDK):
   ```powershell
   winget install Microsoft.VisualStudio.2022.BuildTools
   ```
   Κατά την εγκατάσταση, επιλέξτε τη ροή εργασίας **"Desktop development with C++"**.

2. **Εργαλειοθήκη Rust:**
   ```powershell
   winget install Rustlang.Rustup
   ```
   Μετά την εγκατάσταση, εκτελέστε `rustup default stable`.

3. **Επαλήθευση:**
   ```powershell
   rustc --version
   cargo --version
   ```

#### Προαιρετικά

- **Docker Desktop** — απαιτείται μόνο εάν χρησιμοποιείτε το [Docker sandboxed runtime](#υποστήριξη-περιβάλλοντος-εκτέλεσης-τρέχουσα) (`runtime.kind = "docker"`).

</details>

<details>
<summary><strong>Linux / macOS</strong></summary>

#### Απαιτείται

1. **Βασικά εργαλεία κατασκευής:**
   - **Linux (Debian/Ubuntu):** `sudo apt install build-essential pkg-config`
   - **Linux (Fedora/RHEL):** `sudo dnf group install development-tools && sudo dnf install pkg-config`
   - **macOS:** Εγκαταστήστε τα Xcode Command Line Tools: `xcode-select --install`

2. **Εργαλειοθήκη Rust:**
   ```bash
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```

3. **Επαλήθευση:**
   ```bash
   rustc --version
   cargo --version
   ```

#### Πρόγραμμα εγκατάστασης μίας γραμμής

Ή παρακάμψτε τα παραπάνω βήματα και εγκαταστήστε τα πάντα με μία εντολή:

```bash
curl -LsSf https://raw.githubusercontent.com/zeroclaw-labs/zeroclaw/main/scripts/install.sh | bash
```

#### Απαιτήσεις πόρων μεταγλώττισης

Η κατασκευή από την πηγή απαιτεί περισσότερους πόρους από την εκτέλεση του δυαδικού αρχείου:

| Πόρος | Ελάχιστο | Συνιστώμενο |
|---|---|---|
| **RAM + swap** | 2 GB | 4 GB+ |
| **Ελεύθερος δίσκος** | 6 GB | 10 GB+ |

Εάν το σύστημά σας είναι κάτω από το ελάχιστο, χρησιμοποιήστε προκατασκευασμένα δυαδικά αρχεία:

```bash
./bootstrap.sh --prefer-prebuilt
```

#### Προαιρετικά

- **Docker** — απαιτείται μόνο εάν χρησιμοποιείτε το [Docker sandboxed runtime](#υποστήριξη-περιβάλλοντος-εκτέλεσης-τρέχουσα) (`runtime.kind = "docker"`).

</details>

```bash
# Βασικές εντολές

zeroclaw agent                 # Διαλογική λειτουργία
zeroclaw status                # Έλεγχος κατάστασης
zeroclaw doctor                # Διαγνωστικά συστήματος
```

## Αρχιτεκτονική

Κάθε υποσύστημα είναι ένα **trait** — αλλάξτε την υλοποίηση με μια αλλαγή στη ρύθμιση, χωρίς αλλαγές στον κώδικα.

| Υποσύστημα | Trait | Περιλαμβάνει |
|-----------|-------|------------|
| **Μοντέλα AI** | `Provider` | OpenAI, Anthropic, Ollama, vLLM, κ.α. |
| **Κανάλια** | `Channel` | CLI, Telegram, Discord, Slack, WhatsApp, κ.α. |
| **Μνήμη** | `Memory` | SQLite, PostgreSQL, Markdown, κ.α. |
| **Εργαλεία** | `Tool` | shell, file, git, browser, κ.α. |
| **Ασφάλεια** | `SecurityPolicy` | sandbox, allowlists, encrypted secrets |

## Ασφάλεια

Το ZeroClaw επιβάλλει ασφάλεια σε **κάθε επίπεδο**:
- **Πύλη:** Σύνδεση στο `127.0.0.1` από προεπιλογή.
- **Σύζευξη:** Απαιτείται κωδικός 6 ψηφίων για την πρώτη σύνδεση.
- **Χώρος εργασίας:** Περιορισμένη πρόσβαση μόνο στον κατάλογο του έργου.
- **Λίστες επιτρεπόμενων:** Απόρριψη όλων των εισερχόμενων από προεπιλογή.
